# 🎯 Target Hunt (Unreal Engine Game)

A first-person target hunting game built using Unreal Engine, where players explore a large open-world environment to locate and shoot hidden targets within a limited time.

---

## 🧠 Game Overview

Target Hunt is an **FPP (First Person Perspective)** game where the player is equipped with a gun with unlimited ammo and must locate **25 hidden targets** across a large map.

* ⏱️ **Game Timer:** 25,000 in-game seconds
* ⚡ **Time Scaling:** 1 in-game second = 0.01 real seconds
* ⏳ **Actual Time Limit:** ~250 seconds
* 🎯 Objective: Find and shoot all targets before time runs out

---

## 🗺️ Map Details

* 📏 Map Size: **750 × 750 meters**
* 🎨 Theme: **Chinese / Asian Village**
* 🌿 Environment: Dense structures, obstacles, and exploration-based design

---

## 🎮 Gameplay Features

* First-person shooting mechanics
* Unlimited ammo system
* Hidden targets across environment
* Fast-paced timer-based gameplay
* Smooth performance (~60 FPS at 1080p)
